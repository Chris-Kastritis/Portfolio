PROJECT 1:

DnD Gameboard

DnD Gameboard is a website that will allow users to create, share, and play on Dungeons and Dragons maps with friends from anywhere.

❮ ❯

V0.1

This version is focused on the creation of the main functionality of the grid that will be used as the gameboard.

CHANGELOG:
  • Created the main grid in which the dimensions can be changed to fit the needs of the users
  • Boxes within the grid will change size to fill available space based on the dimensions
  • Added zoom functionality to the grid up to 4x
  • Added drag functionality while zoomed to allow users to move across the board as necessary while keeping the grid within its borders

Future Plans

This wireframe shows a possible design for the future features that will be added to the application. This wireframe will be fleshed out as more features are added and a more clear design starts to shine through.

V.02:
  • Create game piece functionality
  • Allow the creation of enemy and ally game pieces
  • Upload images to create more detailed game pieces
FUNCTIONALITY PRIORITY
  • Upload images to the map grid to create a more immersive experience
  • Allow the creation of maps using walls, doors and other features
  • Create login functionality
  • Allow users to save game states to be used at a later time
  • Allow users to invite other users to their game to allow them to see and interact with it
  • Allow users to set privledges for how other users can interact with their game
  • Improve overall design
